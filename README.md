# Bar Cafetería Los Espejitos

Sitio web oficial del Bar Cafetería Los Espejitos, ubicado en La Laguna, Tenerife.

## 🌐 Enlace al sitio

[losespejitos.replit.app](https://losespejitos.replit.app)

## 📋 Descripción

Página web completa y responsive para el Bar Cafetería Los Espejitos. Incluye:

- **Hero** con fotografía real del local y llamada directa para reservas
- **Sobre Nosotros** — historia y valores del bar
- **Servicios** — todo lo que ofrece el establecimiento
- **Carta interactiva** — menú completo con pestañas por categoría y modal de carta completa
- **Galería** — fotos de platos y ambiente con lightbox
- **Reservas** — botón directo de llamada al 922 821 200
- **Reseñas** — opiniones reales de clientes de Google
- **Mapa y contacto** — dirección, horario y teléfono

## 🛠️ Tecnologías

- **React 18** + **TypeScript**
- **Vite** — bundler y servidor de desarrollo
- **Tailwind CSS** — estilos
- **Lucide React** — iconografía

## 🚀 Despliegue en Vercel

1. Importa este repositorio en [vercel.com](https://vercel.com)
2. Vercel detecta automáticamente la configuración gracias a `vercel.json`
3. Haz clic en **Deploy** — listo en menos de 2 minutos

## 💻 Desarrollo local

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build de producción
npm run build
```
